API de Imágenes

Esta es una API simple para servir imágenes estáticas desde Netlify.

Cómo usar

Para acceder a las imágenes, utiliza la siguiente URL:

https://[tu-sitio].netlify.app/img/nombre-de-imagen.png

También puedes acceder a imágenes SVG:

https://[tu-sitio].netlify.app/img/nombre-de-imagen.svg
