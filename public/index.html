API de Recursos | SenaCloud

Esta es una API simple para servir todo tipo de archivos que requieran.

Cómo usar

Para acceder a las imágenes, utiliza la siguiente URL:

https://[tu-sitio].netlify.app/img/nombre-de-imagen.png

También puedes acceder a imágenes SVG:

https://[tu-sitio].netlify.app/img/nombre-de-imagen.svg

Para acceder a los videos, utiliza la siguiente URL:

https://[tu-sitio].netlify.app/videos/nombre-de-video.mp4
